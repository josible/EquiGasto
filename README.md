# EquiGasto

Aplicación móvil para reparto de gastos similar a Splitwise.

## Arquitectura

La aplicación está construida siguiendo **Clean Architecture** con las siguientes capas:

- **Presentation**: Widgets, pantallas y providers (Riverpod)
- **Domain**: Entidades, use cases y repositorios abstractos
- **Data**: Implementaciones de repositorios y fuentes de datos

## Funcionalidades

- ✅ Autenticación de usuarios
- ✅ Gestión de grupos
- ✅ Gestión de gastos
- ✅ Sistema de deudas
- ✅ Notificaciones

## Tecnologías

- Flutter 3.x
- Riverpod para gestión de estado
- GoRouter para navegación
- Clean Architecture
