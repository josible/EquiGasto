# Assets

## Logo de Google

Para el botón de "Continuar con Google", necesitas el archivo `google_logo.png`.

### Opciones para obtener el logo:

1. **Descargar desde Google Brand Resources:**
   - Visita: https://developers.google.com/identity/branding-guidelines
   - Descarga el logo oficial de Google
   - Redimensiona a 24x24 píxeles o similar
   - Guarda como `google_logo.png` en esta carpeta

2. **Usar un generador de logos:**
   - Puedes usar herramientas online para generar un logo de Google
   - Asegúrate de que sea transparente (PNG con fondo transparente)
   - Tamaño recomendado: 24x24 o 48x48 píxeles

3. **Nota:**
   - Si el logo no está disponible, la app usará un ícono de Material Icons como fallback
   - El botón seguirá funcionando correctamente
